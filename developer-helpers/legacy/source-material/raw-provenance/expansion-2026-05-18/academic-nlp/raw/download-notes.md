# Academic NLP Download Notes

Date: 2026-05-18

Owned folder: `legacy/source-material/expansion-2026-05-18/academic-nlp/`

## Downloaded Or Captured Locally

- WikiSQE: Hugging Face dataset card and API metadata.
- Wiki Neutrality Corpus: GitHub README and repository API metadata. The linked full corpus is 100 MB compressed and about 500 MB expanded, so it was not downloaded.
- Detection of Puffery on English Wikipedia: GitHub README, repository API metadata, data listing, full clean peacock sample file, and clean non-peacock comparator file.
- CoNLL-2010 Hedge Detection / BioScope: ACL shared task page and BioScope corpus page. Full shared-task data requires registration for some material, so no full download was made.
- VAQUUM: GitHub README, repository API metadata, data listing, `images.csv`, and a 120-line sample of `human_ratings.csv`.
- MPQA Subjectivity Lexicon / OpinionFinder: MPQA README and full subjectivity-clues lexicon from the Illinois CogComp mirror. The official `mpqa.cs.pitt.edu` host did not resolve from this environment.
- General Inquirer: official category page and spreadsheet guide. The full dictionary spreadsheet was not downloaded because the site records commercial-use permission constraints.
- SQUINKY: arXiv page and PyPI JSON metadata. No standalone corpus download was found during this extraction.
- SpecificityTwitter: GitHub README, repository API metadata, data listing, and 120-line sample of `cleaned_data.tsv`.
- TAACO: GitHub README and official tool page.
- TAALES: official tool page.
- TAASSC: GitHub README.
- DiMLex-Eng: full `en_dimlex.xml`, repository metadata, and ACL page.
- GUM: GitHub README, repository metadata, and a 220-line UD sample from UD English GUM.
- MAGPIE: GitHub README, repository metadata, content listing, and a 40-line jsonl sample.
- STREUSLE: GitHub README, repository metadata, content listing, and a 220-line conllu sample.

## Inaccessible Or Not Downloaded

- `https://mpqa.cs.pitt.edu/lexicons/subj_lexicon/subjclueslen1-HLTEMNLP05.tff`: DNS resolution failed from this environment. The same lexicon was obtained from an Illinois CogComp mirror.
- CoNLL-2010 shared task extra biomedical articles: access path points to registration.
- Wiki Neutrality Corpus full zip: not downloaded because it is large relative to this extraction task.
- MAGPIE full jsonl files: not downloaded because each file is about 45 to 55 MB and samples were enough for metadata and fixture planning.
- General Inquirer full dictionary: not downloaded because the official site distinguishes research and commercial permission.
- SQUINKY standalone corpus: no direct corpus download was found; PyPI metadata and the paper page were captured instead.
