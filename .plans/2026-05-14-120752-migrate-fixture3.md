# Goal

Migrate active textlint fixture approval from `goldencheck` to `fixture3`.

The migration must preserve approved normalized output and fixture behavior. Only the active runner surface changes.

# Approach

1. Rename `goldencheck.yaml` to `fixture3.yaml`.
2. Move runtime received/diff storage from `.goldencheck/<suite>` to `.fixture3/<suite>`.
3. Replace active wrapper `scripts/goldencheck.sh` with `scripts/fixture3.sh`.
4. Update active verifiers to read `fixture3.yaml`, validate `.fixture3`, and run `scripts/fixture3.sh`.
5. Keep approved output under `behavior/golden/<suite>`.
6. Retire active goldencheck verifier naming where it is part of the current runner surface.
7. Keep historical plans and old prose untouched unless they are active verification inputs.
